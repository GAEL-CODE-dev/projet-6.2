# Portfolio — Gaël Yangu

Site portfolio personnel d'un apprenant développeur Full-Stack en formation chez **Akienie Academy**.

## Technologies

- HTML5
- CSS3 (variables, flexbox, grid, animations, backdrop-filter, content-visibility)
- Aucun JavaScript — interactivité purement CSS (checkbox hack)

## Fonctionnalités

- Mode sombre / clair (toggle CSS-only)
- Menu hamburger responsive (CSS-only)
- 12 compétences avec barres animées
- 6 projets avec overlay au survol
- Timeline du parcours
- Formulaire de contact
- Accessibilité : skip-link, labels, aria-hidden, prefers-reduced-motion

## Structure

```
.
├── index.html    # Page principale (sections: hero, à propos, compétences, projets, parcours, contact)
├── style.css     # Styles complets (mobile-first, dark mode, animations)
└── README.md
```

## Utilisation

Ouvrir `index.html` dans un navigateur. Aucune installation requise.
